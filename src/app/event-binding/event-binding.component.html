Event Binding

In the event-binding component class, we created a public property welcome to use as an empty string. This string will be filled when we call the method displayWelcome(), which will be called when the user clicks a button:
public welcome = "";

displayWelcome() {
this.welcome = "Welcome to the 'Learning Angular App'";
}

First, we will create a button with event binding, (click), then use it to call the displayWelcome() method.
HTML: <button (click)="displayWelcome()">Click to Display</button>

We will then use interpolation:
HTML: <h4>{{welcome}}</h4>
