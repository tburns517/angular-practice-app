Style Binding

The style-binding class contains a public boolean property, hasError, which is set to true. We can use this property to determine the color that will be applied to the header below.

Consider this HTML for the header:
<h4 [style.color]="hasError ? 'red' : 'green'">

Since we know hasError is true, the color 'red' will be applied:

This header should be red

As you can see, the color 'red' is applied. If there isn't an error, the color 'green' should be applied to the header. We can try this by creating a new property, otherHasError, and setting it to false, then replacing hasError.

Consider this HTML for the header:
<h4 [style.color]="otherHasError ? 'red' : 'green'">

This header should be green

We can also bind the style directly using a property. Next, we will apply the color 'blue' to the header by setting the public property, colorBlue, equal to "blue".

Consider this HTML for the header:
<h4 [style.color]="colorBlue">

This header should be blue

Similarly to the above method of binding, we can create an object to apply multiple styles to the header using the ngStyle directive. In style-binding component class, the headerStyles object was created:
public headerStyles = {
color: "orange",
fontStyle: "italic"
}

Consider this HTML for the header:
<h4 [style.color]="headerStyles">

This header should be orange and italicized
