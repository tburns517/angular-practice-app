Two-Way Binding

In order to add two-way binding we will use the ngModel directive. First, we have added a public property name as an empty string in the two-way-binding component class. You can view how we use this property in the HTML below:
<input [(ngModel)]="name" type="text">

Underneath the input field, we will use interpolation: {{name}}

See the results:


{{name}}
IMPORTANT!: Remember to add the FormsMoulde to app.module.ts in the 'imports' section, or ngModel will not be recognized.
